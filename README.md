# Customer Churn & Revenue Analytics

## Project Overview
This project analyzes customer churn behavior using real-world styled telecom data.
The goal is to identify churn drivers, revenue loss, and high-risk customers.

## Dataset
- Source: Kaggle (Customer Churn Dataset)
- Records: 7,044 customers
- Features: Demographics, services, contract type, tenure

## Tools & Technologies
- Python (Pandas, NumPy)
- MySQL (XAMPP)
- SQL
- Jupyter Notebook
- Data Visualization

## Key Analysis
- Churn rate by contract type
- Revenue contribution of churned vs retained customers
- Payment failure impact on churn
- High-risk customer segmentation

## Business Insights
- Month-to-month contracts show highest churn
- Customers with payment failures have significantly higher churn risk
- Early-tenure customers contribute to major revenue loss

## Outcome
This analysis can help companies reduce churn and improve customer lifetime value.
